# Web UI

@../CLAUDE.md

Browser-based reactive interface for the Chladni pattern generator.

## Files

- `index.html` -- single-page app structure
- `style.css` -- dark theme, polished controls
- `app.js` -- canvas rendering, marching-squares contour, magma colormap, Lorentzian mode weighting, on/off flatten/resolve animation
- `data.js` -- generated by `../build_web.py`. Contains the precomputed mode bank for the default steel plate (square 20cm × 1mm, circle radius 10cm × 1mm). Update by re-running the build script.

## Run

Either open `index.html` directly in a browser, or:

```
cd .. && python -m http.server
# then visit http://localhost:8000/web/
```

## Architecture

- **Square plate:** the JS computes ψ(x,y) per frame from the mode list (just (n, m) tuples + frequencies). cos products are separable and very fast.
- **Circular plate:** ψ_{n,m}(x,y) is precomputed in `build_web.py` and stored as base64 Float32 fields in `data.js`. Per frame, the JS sums fields with Lorentzian weights.
- **Mode weighting:** `w_n(f) = 1 / ((f² − f_n²)² + (γf)²)` with `γ = f / Q`. Higher Q = sharper snap to mode peaks. Default Q = 30. Slider exposes Q from 5 (very soft) to 200 (sharp).
- **On/off animation:** the amplitude state is animated toward `target ∈ {0, 1}` with an exponential settle. Asymmetric time constants: `τ = 0.18s` ramping up (resolve), `τ = 0.45s` ramping down (decay). Visually the pattern crystallizes faster than it dissipates.

## Update mode bank

If `chladni.py` defaults change or you want a different plate:

```
python build_web.py
```

Re-emits `web/data.js`.
