## Build provenance

Generated by `../scripts/build_charts_data.py` from existing source CSVs in the task root and `../followups/`. No new SQL was run; this is a projection/aggregation/join layer over already-verified query outputs.

## Conventions

- Long format where the chart shows multiple series over time on one axis (Chart 1, Chart 2, Chart 4 expressed as either form per chart-tool preference)
- Wide format where each column is a distinct series sharing an x-axis (Chart 3, Chart 5, Chart 6)
- All dates as `YYYY-MM-DD` first-of-month
- Rates as decimals (0.20 = 20%); percentage formatting belongs in the chart, not the data
- No formatting characters (no `$`, `%`, `,`)
