# Data Health

@../CLAUDE.md

Investigations into data quality, tracking integrity, and metric reliability.

## Conventions

- Each investigation lives in a date-stamped task folder: `YYYY-MM-DD-<slug>/`
- Use `signal-detection.md` or `data-quality-report.md` templates as appropriate
- Include all diagnostic SQL in `visitors.sql` within the task folder
- Document root cause and remediation feasibility for each finding
