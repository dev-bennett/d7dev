## Program Scope

The lifecycle email program spans 6 macro-stages:

1. **Acquire** -- Free account onboarding, MQL/PQL nurture, trial conversion
2. **Activate** -- New subscriber onboarding to first value moment
3. **Engage** -- Deepen habitual usage (personalized digests, cross-library discovery)
4. **Retain** -- At-risk intervention (existing 5-segment model lives here)
5. **Expand** -- Upsell, cross-sell, monthly-to-annual, enterprise AE handoff
6. **Winback** -- Churned subscriber re-acquisition

Cross-cutting: enterprise lifecycle track, in-app event triggers, suppression/orchestration, engagement scoring, measurement framework.
