# PQL Tasks

@../CLAUDE.md

Container for dated PQL analysis tasks. Each task lives in `YYYY-MM-DD-<slug>/` with its own CLAUDE.md and purpose statement.

## Conventions

- Task directory naming: `YYYY-MM-DD-<descriptive-slug>/`
- Each task directory has its own CLAUDE.md (no exceptions per guardrails.md §Directory CLAUDE.md Chain)
- Task-level FINDINGS.md aggregates results; per-query-set FINDINGS.md live inside their query-set subdirectories
- Closed tasks remain in place (do not archive); outcomes get promoted to knowledge/ or a memory entry
