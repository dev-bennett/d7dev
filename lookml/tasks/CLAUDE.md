# LookML Tasks

@../CLAUDE.md

Task-based work directories for LookML development initiatives.

## Conventions

- Each task gets a dated directory: `YYYY-MM-DD-<slug>/`
- Every task directory must have CLAUDE.md and README.md
- LookML source files go in `lkml/` subdirectory mirroring repo structure
- Check existing tasks before starting new work on the same explores/views
